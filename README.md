# AI & Machine Learning Projects

A collection of Machine Learning projects built using Python, Scikit-learn, Pandas and Matplotlib.

## Projects

### 1. Spam Email Detector
- Classifies emails as spam or not spam
- Algorithm: Naive Bayes
- Dataset: SMS Spam Collection (5,572 messages)
- Accuracy: 97.85%

### 2. Rainfall Prediction Model
- Predicts whether it will rain tomorrow
- Algorithms: Logistic Regression and Decision Tree
- Dataset: Australian Weather Dataset (65,298 records)
- Accuracy: 83.37%

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
